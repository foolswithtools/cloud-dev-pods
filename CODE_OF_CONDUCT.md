# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, in full. The canonical text is published at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (issues, pull requests, discussions, code review), you agree to abide by it.

## Reporting

Concerns about behavior that violates the Code of Conduct can be reported to the maintainers privately via:

- GitHub's [private vulnerability reporting](https://github.com/foolswithtools/cloud-dev-pods/security/advisories/new) (also used for security; mark the report as a Code of Conduct concern in the body), or
- A direct message to a maintainer listed in [`.github/CODEOWNERS`](.github/CODEOWNERS).

All reports will be reviewed and investigated promptly and confidentially.

## Enforcement

Enforcement follows the Contributor Covenant 2.1 enforcement guidelines, including the four-tier consequence ladder (Correction → Warning → Temporary Ban → Permanent Ban). The maintainer team has discretion over the appropriate response to any incident.
